# 🚀 PromptHub Free 当天上线手册

恭喜！你拿到的是一套专门为**零基础、零成本、快速变现**设计的精简版系统。请严格按照以下步骤操作，30 分钟内你的网站就能上线。

---

## 🛠️ 第一阶段：环境准备 (10 分钟)

1.  **注册 Firebase**
    *   访问 [Firebase Console](https://console.firebase.google.com/)。
    *   点击 "Add project"，起名为 `prompthub-yourname`。
    *   **重要**：无需绑定信用卡，选择默认的 Spark Plan（免费）。
2.  **创建 Web 应用**
    *   在 Firebase 项目主页点击 `< >` 图标（Web）。
    *   注册 App 后，你会看到一段 `firebaseConfig` 代码。
    *   **打开 `public/js/firebase-config.js`**，把这段代码粘贴进去。

---

## 🎨 第二阶段：个性化修改 (10 分钟)

1.  **替换收款码**
    *   准备你的微信支付或支付宝收款二维码。
    *   将图片命名为 `wepay-qr.png`，放在 `public/images/` 目录下。
2.  **修改联系方式**
    *   打开 `public/pay.html`。
    *   将 `your_wechat_id` 替换为你真实的微信号。
3.  **配置 SEO（可选）**
    *   打开 `public/index.html`。
    *   修改 `<title>` 和 `<meta description>` 为你自己的品牌名。

---

## 🚢 第三阶段：一键发布 (5 分钟)

推荐使用 **GitHub + Firebase Hosting** 自动部署：

1.  **新建 GitHub 仓库**：起名 `prompthub-free`。
2.  **本地上传代码**：
    ```bash
    git init
    git add .
    git commit -m "init"
    git remote add origin [你的仓库地址]
    git push -u origin main
    ```
3.  **安装 Firebase 工具**：
    ```bash
    npm install -g firebase-tools
    firebase login
    firebase init hosting
    ```
    *   选择你的项目 ID。
    *   What do you want to use as your public directory? 输入 `public`。
    *   Configure as a single-page app? 输入 `No`。
    *   Set up automatic builds and deploys with GitHub? 输入 `Yes`（这步最关键，以后你改代码 Push，网站自动更新）。
4.  **执行部署**：
    ```bash
    firebase deploy
    ```

---

## 💰 第四阶段：开始搞钱 (5 分钟)

1.  **首单测试**：
    *   打开你的域名（如 `https://xxx.web.app`）。
    *   点击一个 Prompt，测试支付流程，提交订单。
2.  **小红书冷启动**：
    *   **封面图**：截取网站中“MJ商业摄影”或“小红书爆款生成器”的精美界面。
    *   **文案**：标题《别再找了！这50个宝藏AI提示词库，建议收藏！》。
    *   **引流**：在评论区置顶“获取方式见个人简介”或直接发链接。

## 💡 给小白的建议

*   **关于发货**：目前是人工模式。当有人在 `pay.html` 提交订单后，你会在 Firebase 控制台（如果你配置了 Firestore）看到，或者通过微信私域直接沟通发货。
*   **关于扩容**：如果你赚到了第一个 1000 元，建议购买一个 `.com` 域名（约 60 元/年）绑定到 Firebase，专业感倍增。
*   **关于内容**：每天花 10 分钟，把你自己觉得好用的 Prompt 加进 `public/data/prompts.json`。
